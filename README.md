# aardio-ScottPlot
aardio调用c# ScottPlot 控件显示波形图,ScottPlot是用c# .net编写的波形图控件, 在winforms上性能优异, 上手简单, 只需要几行代码即可生成漂亮丝滑的波形图. 支持 折线图 / 直方图 / 饼图 / 散点图, 等等

![image](https://github.com/popde/aardio-ScottPlot/blob/main/pic2.png)
